TrackOut is a an health tracking app, with the aim to assist you daily. We want to remind you of your goals and help you achieve them.

## 📚 Getting started

To get up and running:

1. Clone the repo
2. Install dependencies
   ```
       yarn
       npx pod-install
   ```
3. Run the app
   ```
       yarn ios
   ```
